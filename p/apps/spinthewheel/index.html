Description

Having a hard choice? Spin the wheel to decide! This app helps you make tough decisions, like what to eat or what to do when you're stuck.

Features

  • Add new categories for different choices
  • Add options to each category
  • Disable options temporarily by unmarking them

Two Different Modes

  • Normal mode: Each option has the same probability and each slice is equally sized.
  • Party mode: Each option has the same probability, but not all slice sizes are equally large to spice things up!

Note: This app is mostly a joke between me and my friends, but it works well enough to be put on the Google Play Store.
